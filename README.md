# Nutrition Tracker

Single-file local nutrition tracker prepared for GitHub Pages.

## Local files

- Website entry point: index.html
- Original uploaded file: nutrition-tracker.html
- GitHub Pages workflow: .github/workflows/pages.yml

## Publish

1. Create a new GitHub repository named nutrition-tracker.
2. Push this folder to the repository main branch.
3. In GitHub, open Settings > Pages and set Source to GitHub Actions.
4. The Deploy GitHub Pages workflow will publish the site.
